# Dotfiles & GNU Stow

## How to set up

1. Install [GNU Stow](https://www.gnu.org/software/stow/)
```bash
brew install stow
```
2. Clone this repo to the home folder
```bash
git clone "git@github.com:pr0f37/dotfiles.git"
```
3. Enter the `~/dotfiles` folder
```bash
cd ~/dotfiles 
```
4. Execute stow command
```bash
stow .
```
5. Done
